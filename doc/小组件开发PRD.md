# 小组件主要功能
核心目的：用户展示设置里选中的答案库数据，小组件每天随机更新展示其中的一条数据。

## UI色调
1.小组件背景遵循iOS 26的规范，支持各种主题，比如黑色，白色透明等。
2.小组件背景采用灰白色，且可以跟随系统主题适当调整。

### UI布局
1.小组件上方显示今天日期年月日比如：2025-09-25 
2.中间显示答案库数据，每天随机更新展示其中的一条答案数据

### 小组件数据更新策略
1.小组件获取答案需跟随用户在设置页面里选中的答案库，如果用户在设置里修改了答案库小组件显示的内容要同步更新，小组件数据更新策略，请遵循iOS小组件开发规范。

2.小组件展示的数据每10秒钟，刷新一次，重新选择一条答案更新。
